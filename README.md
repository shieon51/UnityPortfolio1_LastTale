# LAST TAIL

LAST TAIL은 **2D 액션 RPG + 어드벤처** 기반의  
**멀티 루트 · 멀티 엔딩 내러티브 게임**입니다.

(개발: 현재 진행중)

플레이어의 선택, 시간 사용 방식, 반복 플레이 기록이  
스토리와 엔딩 분기에 직접적인 영향을 줍니다.  
단일 플레이로 모든 콘텐츠를 경험할 수 없으며,  
회차를 거듭할수록 새로운 선택지와 진실에 접근하게 됩니다.

---

## Core Concept
- 장르: 2D Action RPG / Adventure 
- 구조: 멀티 루트 · 멀티 엔딩
- 핵심 요소: 시간 관리, 선택의 누적, 회차 기반 변화

(초안)
총 5개의 기본 루트를 중심으로 스토리가 전개되며,  
각 루트마다 3개의 엔딩이 존재합니다.  
모든 기본 루트의 엔딩을 확인하면  
히든 루트(사실상의 진엔딩)가 해금되는 구조를 구상 중입니다.

---

## Key Systems

### Time Coin System
- 하루 시작 시 24개의 시간 코인 지급
- 모든 행동은 시간 코인을 소모
- 사용된 시간에 따라 현재 시각이 결정
- 하루에 주어진 모든 이벤트를 경험할 수 없음

→ 시간 선택이 곧 스토리 분기의 조건이 됩니다.

---

### Fatigue System
- 활동 시간이 길어질수록 피로도 누적 (수면 필요)
- 피로도 증가 시 능력 효율 감소 또는 강제 수면
- 수면 시간에 따라 회복량 차등 적용

---

### Time-based Events
- 시간대에 따라 NPC 위치 및 대화 변화
- 특정 이벤트는 특정 시간대에만 발생

---

### Loop-based Events (회차 시스템)
- 엔딩 이후 재시작 시 이전 기록 반영
- 지난 회차의 엔딩 or 지난 회차에서 얻은 정보에 따라 선택지, 대화, 이벤트 변화
- 이전 회차에서 얻은 정보 활용 가능

---

### Save / Load Limitation
- 세이브 및 시간 회귀 시 게임 내 MP 소모
- 중간 세이브 또한 MP 소모
- 세이브를 캐릭터의 고유 능력으로 설정
